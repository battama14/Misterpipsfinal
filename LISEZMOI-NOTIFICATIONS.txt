🚀 COMMENT ACTIVER LES NOTIFICATIONS (3 ÉTAPES)
════════════════════════════════════════════════

1️⃣ OUVRIR L'APPLICATION
   • Ouvrir mobile-dashboard.html sur votre mobile

2️⃣ ALLER DANS PARAMÈTRES
   • Menu ☰ → Paramètres → Notifications Chat

3️⃣ ACTIVER
   • Cliquer sur "🔔 Activer les Notifications"
   • Accepter les permissions dans la popup
   • Une notification de test s'affiche
   • Personnaliser les toggles selon vos préférences

C'EST TOUT ! 🎉

🐛 PROBLÈMES COURANTS
═════════════════════

❌ Les notifications ne s'affichent pas
   → Vérifier que les permissions sont accordées
   → Vérifier que le toggle "Push" est activé
   → Quitter l'écran du chat (notifications ne s'affichent pas sur le chat)
   → Consulter : GUIDE-NOTIFICATIONS-UTILISATEUR.md (Section Dépannage)

❌ Le son ne se joue pas
   → Vérifier que le toggle "Son" est activé
   → Vérifier le volume du téléphone
   → Désactiver le mode silencieux
   → Consulter : README-NOTIFICATIONS.md (Section Dépannage)

❌ Les paramètres ne se sauvent pas
   → Vérifier qu'il n'y a pas d'erreur JavaScript (console F12)
   → Effacer le cache du navigateur
   → Consulter : DEPLOIEMENT-NOTIFICATIONS.md (Section Résolution)
